🏥 GuptaCare Pharmacy Website

A modern **Pharmacy & Medical Website** developed as part of the **Web Technology Lab Project**.  
This project provides an online platform for medicine browsing, cart management, doctor appointment booking, and order placement.

---

 🌐 Live Preview
👉 https://ashishgupta321.github.io/guptacare-pharmacy-website/

---

📌 Project Features

🛒 Online Pharmacy
- Browse medicines by category (Tablets, Syrups, First Aid, Vitamins, Personal Care)
- Best-selling medicines section
- Add to Cart with quantity management
- Cart count indicator

 💳 Cart & Checkout
- Dynamic cart using **LocalStorage**
- Order summary with subtotal & shipping charges
- Multiple payment options:
  - Cash on Delivery
  - UPI
  - Card (Demo)

 👨‍⚕️ Doctor Consultation
- Doctor profiles with specialization
- Visiting days & timings
- Appointment booking popup
- Appointment data stored (PHP file-based)

 📞 Contact System
- Contact form for customer queries
- Messages stored on server

 🎨 UI & Design
- Clean, professional medical theme
- Responsive layout
- Hover effects & smooth UI
- Pharmacy-friendly green color palette
